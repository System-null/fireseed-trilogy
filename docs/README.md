# Documentation

This folder collects human-facing docs for the **Fireseed Trilogy — Machine-readable Civilization Framework**.

## What's here
- `books/` — canonical book files (EN/ZH) for the trilogy.
- `release-notes/` — versioned change logs for repository releases.
- `ACCESS_TERMS.md` — plain-language guidance for sharing permissions.
- `CITING.md` — how to cite, with DOI.
- `quickstart.en.md` / `quickstart.zh.md` — 10‑minute setup.
- `books/metadata.yml` — ISBN/ASIN mapping and Amazon links.
