# Fireseed Access Terms · 常见条款说明

> 本文档是给非技术用户的简明参考，不构成法律意见。若涉及法律效力，请咨询专业人士。

## 选项速览
- **仅可阅读（read_only）**：他人可以阅读，但不能复制、改写或二次分发。
- **需署名引用（by_attribution）**：他人可以使用你的内容，但必须署名你。
- **可用于非商业研究（non_commercial）**：允许学术或非商业用途；禁止商业销售或广告使用。
- **自定义条款（custom）**：你可以写「仅限家人」「仅限指定联系人」「CC BY-NC 4.0」等。

## 建议
- 若不确定，保持默认「未声明」。未来你可以重新生成并补充条款。
- 若选择「自定义」，尽量**具体**：允许什么、禁止什么、适用范围是谁、有效期多久。
