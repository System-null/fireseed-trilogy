Fireseed Capsule Generator v3.7
--------------------------------
这是一款离线运行的文明接口工具。
你可以在 3 分钟内，生成一份机器可读的人类火种。

用法：
1. 打开 public/generator.html
2. 填写表单（可留空）
3. 点击“生成胶囊”
4. 下载 YAML 或 打印 PDF；可选上传 IPFS（留空也可）

说明：
- 纯前端运行，不上传任何数据
- YAML + PDF 多格式导出
- 兼容 Fireseed Schema v0.3.x

法律提示：
本工具仅用于记录思想与意愿，不具备法律效力。
